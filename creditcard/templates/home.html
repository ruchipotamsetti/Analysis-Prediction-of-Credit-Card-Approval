Analysis and Prediction of Credit Card Approval

It is a project developed in order to help banks shortlist the list of suitable candidates for credit card approval from the hundreds of applications that they receive. Credit card approval predictor analyzes user input and using an optimum machine model will decide whether the candidate should be approved or not. Banking industries receive so many applications for credit card request. Going through each request manually can be very time consuming, also prone to human errors. However, we can use the historical data to build a model which can shortlist the candidates for approval.
